# To-Do List App

This is a simple to-do list app built with HTML, CSS, and JavaScript. It allows you to add, edit, and delete tasks, as well as mark them as completed.

## Features

- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks
- Clear all tasks
- Save tasks to local storage
